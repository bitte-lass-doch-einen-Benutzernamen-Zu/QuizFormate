# Bilder für Quizfragen

Lege Bilder und Audiodateien für Fragen in diesem Ordner ab.

Beispiel:

```text
public/images/questions/person-01.jpg
```

Der öffentliche Pfad innerhalb der App lautet dann:

```text
/images/questions/person-01.jpg
```

Empfehlungen:

- Dateinamen ohne Leerzeichen verwenden
- Kleine Buchstaben und Bindestriche verwenden
- Formate: `.jpg`, `.png`, `.webp` oder `.svg`
- Audioformate: vorzugsweise `.mp3`, alternativ `.ogg` oder `.wav`
- Für Beamer möglichst Bilder ab 1280 × 720 Pixel verwenden
- Große Dateien vorab komprimieren
